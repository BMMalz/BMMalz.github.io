  • Python Wing IDE for Windows, Mac OS X, and Linux:
    First install Python 3.6. Then go to Wingware IDE 101. Download the appropriate package for your computer/operating system. This is the free version of the IDE, and is the same version that is installed in the Wheaton lab.

    Note for Mac users: - you may need to update Wing to use Python 3 (you only need to do this once):
    1. Start Wing
    2. Go to Edit->Configure Python. You'll get a window that looks similar to this.
    3. Change the Python Executable to Custom and type in: /usr/local/bin/python3
    4. Near the bottom where it says "Initial Directory", choose Use default
    5. Restart the shell (or restart Wing)
